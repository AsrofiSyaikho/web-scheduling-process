# Web Based CPU Scheduling Simulator

## Technologies Used
* Flask: A lightweight Python web framework used to handle backend logic, routing, and API development.
* Javascript: The primary scripting language used to create interactive elements and manage dynamic content on the frontend.
* Native CSS: Used to design the visual layout, typography, and responsive styling of the application without external libraries.

## Installation

## Step 1 : Clone The Repository
```bash
git clone https://github.com/AsrofiSyaikho/web-scheduling-process
cd web-scheduling-process
```
## Step 2 : Installing Flask

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install flask.

```bash
pip install flask
```

## Step 3 : Running The App

```bash
python App.py
```
